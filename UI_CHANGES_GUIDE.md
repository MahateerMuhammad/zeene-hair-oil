## ğŸ¬ User Flow Examples

### Adding Product to Wishlist
```
1. Browse products
2. See heart icon on each product
3. Click heart â¤ï¸
   - If logged in: Adds to wishlist
   - If not logged in: Redirects to login
4. Heart turns red and fills
5. Toast notification: "Added to wishlist"
6. Go to Profile â†’ Wishlist to see saved items
```

### Writing a Review
```
1. Buy a product
2. Go to product detail page
3. Click "Reviews" tab
4. Click "Write a Review" button
5. Modal opens
6. Select star rating (1-5)
7. Enter title (optional)
8. Write review text
9. Click "Submit Review"
10. Shows message: "Review submitted! Will appear after approval"
11. Admin approves in backend
12. Review appears on product page
```

### Filtering Products
```
1. Go to /products
2. Desktop: See sidebar on left
   Mobile: Click "Filters" button
3. Select category (e.g., "Hair Care")
4. Products instantly filter
5. Set price range: Min $0, Max $500
6. Products filter again
7. Sort by "Highest Rated"
8. See filtered, sorted results
```
